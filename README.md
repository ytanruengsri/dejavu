# DejaVu

automated page testing using testcafe/nightwatch.js

## Installation
```
$ yarn
```

## How to run testcafe?

with Chrome
```
$ npm run testcafe:chrome
```

with Firefox
```
$ npm run testcafe:firefox
```

## How to run nightwatch?

with Chrome
```
$ npm run nightwatch:chrome
```

with Firefox
```
$ npm run nightwatch:firefox
```
